# ML-kaggle-challenge
## Project for Machine Learning University Course
<br>

The problem is to predict prices of real estates based on series of parameters describing them.

In this project, many subjects are involed including:
- data analysis
- linear regression
- regularization
- decision trees
- grid search
- SHAP analysis
